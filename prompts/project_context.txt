
# AutoCaption Studio - Project Context

## 1. Project Overview

### Project Name

AutoCaption Studio

### Description

AutoCaption Studio is a professional desktop/web application that automatically generates subtitles from videos using AI, allowing users to edit captions, synchronize them with the video, customize subtitle styles, and export high-quality captioned videos.

The application is inspired by CapCut's subtitle editor but runs entirely locally, ensuring privacy, high performance, and no cloud dependency.

---

# 2. Main Goals

The application should allow users to:

* Upload videos
* Automatically generate subtitles
* Edit subtitles
* Edit word timings
* Customize subtitle appearance
* Preview subtitles in real time
* Export SRT files
* Export videos with burned subtitles
* Work completely offline

# 12. Development Rules

Always:

* Keep business logic in Laravel Services
* Keep controllers thin
* Use Repository Pattern if needed
* Use Form Requests for validation
* Use React Query for API
* Use Zustand for global state
* Never call Whisper directly from React
* Laravel is the gateway between Frontend and AI
* Store timestamps in decimal(10,3)
* Generate subtitles as JSON before exporting
